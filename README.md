# Lifegear MVP (Vercel-ready scaffold)

This is a ready-to-deploy React + Vite + TypeScript + Tailwind scaffold that matches your ChatGPT Canvas imports:

- `@/*` alias is configured
- Minimal `src/components/ui/*` exist (Badge/Button/Card/Input/Textarea)

## 1) Paste your Canvas code
Replace `src/App.tsx` with the full code from ChatGPT Canvas.

## 2) Run locally
```bash
npm install
npm run dev
```

## 3) Push to GitHub
Upload the whole `lifegear-mvp/` folder contents into a repo.

## 4) Deploy on Vercel
Vercel → New Project → Import your repo → Deploy

Build command: `npm run build`  
Output directory: `dist`
